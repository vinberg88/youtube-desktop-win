# Privacy Policy for TubeDesk for Windows

Last updated: 2026-05-11

TubeDesk for Windows is an unofficial desktop wrapper for YouTube and YouTube Music.

## Data collection

TubeDesk for Windows does not collect, sell, share, or transmit personal data to the developer.

## Local app data

The app uses Electron's persistent browser session to keep YouTube and Google login state on the user's own device. This may include cookies, cache, and other local browser storage created by YouTube or Google inside the embedded browser session.

This data stays on the user's device unless YouTube, Google, or the embedded web content transmits it as part of their own services.

## Third-party services

TubeDesk displays YouTube and YouTube Music web content. Use of those services is governed by Google's and YouTube's own terms and privacy policies.

## Login

Google/YouTube login is handled directly by Google/YouTube inside the embedded browser session. TubeDesk does not receive, store, or process usernames, passwords, authentication tokens, or payment information.

## Clear local session

The app includes a local session clearing feature. This removes local YouTube/Google session data stored by the app on the device.
