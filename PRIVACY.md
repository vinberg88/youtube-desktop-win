# Privacy Policy for TubeDesk

Last updated: 2026-06-04

TubeDesk is an independent Windows desktop workspace for user-selected web-based video, audio, learning, and live-content workflows. It provides desktop convenience features around web pages chosen by the user, including local shortcuts, mini-player mode, focus mode, always-on-top mode, tray support, mute and zoom controls, and local session controls.

## Data collection

TubeDesk does not collect, sell, share, or transmit personal data to the developer.

## Local app data

TubeDesk uses Electron's persistent browser session to keep local browsing state on the user's own device. This may include cookies, cache, local storage, saved local shortcuts, and other browser storage created while the user opens websites inside the app.

This data stays on the user's device unless a user-selected website or embedded web content transmits it as part of that website's own service.

## Third-party services

TubeDesk can display web content from websites selected by the user. Those websites and services are provided by their respective owners. Use of any third-party website is governed by that website's own terms, account requirements, and privacy policy.

TubeDesk is not affiliated with, endorsed by, sponsored by, or approved by any third-party website, media service, or platform.

## Login

If a user signs in to a third-party website inside TubeDesk, that login is handled directly by the third-party website inside the embedded browser session. TubeDesk does not receive, store, or process usernames, passwords, authentication tokens, or payment information.

## Clear local session

TubeDesk includes a local session clearing feature. This removes local workspace browsing data stored by the app on the device.
