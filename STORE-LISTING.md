# Microsoft Store Listing Draft

Use this draft as the English-only Microsoft Store listing for TubeDesk.

## App name

TubeDesk

## Short description

A focused Windows workspace for user-selected web video, audio, learning, and live-content workflows.

## Long description

TubeDesk is an independent desktop workspace for people who use web-based media and live-content tools in a dedicated Windows window instead of mixing them with regular browser tabs.

The app lets users open the websites they choose and adds desktop-specific workflow features around that experience: saved local shortcuts, mini-player mode, focus mode, always-on-top mode, system tray behavior, mute and zoom controls, keyboard shortcuts, external-browser handoff, and local session clearing.

TubeDesk does not provide, own, or represent any third-party media service. User-selected websites and services are provided by their respective owners and may require their own accounts, terms, subscriptions, regional availability, and privacy policies.

This app is not a downloader, ad blocker, media extractor, scraping tool, DRM circumvention tool, or paid-service bypass tool. It does not modify, scrape, bypass, or circumvent third-party services.

## Key features

| Feature | Description |
|---|---|
| Dedicated desktop workspace | Open user-selected web media and live-content pages in a separate Windows app window. |
| Local custom shortcuts | Save frequently used pages locally on the device for quick access. |
| Mini-player mode | Open the current page in a compact always-on-top window. |
| Focus mode | Hide TubeDesk navigation and tools for a cleaner viewing workspace. |
| Always-on-top mode | Keep the workspace visible above other windows when needed. |
| System tray support | Keep TubeDesk available from the Windows notification area. |
| Mute and zoom controls | Adjust sound and page scale at the app-window level. |
| External-browser handoff | Open the current page in the user's default browser. |
| Local session clearing | Remove locally stored workspace browsing data from the device. |

## Distinct value statement

TubeDesk adds desktop-specific workflow value around user-selected web media usage. Its distinct value is the combination of a dedicated Windows workspace, local custom shortcuts, mini-player workflow, focus mode, tray behavior, persistent local session handling, and window-level controls that are not normally available as a single workflow in ordinary browser-tab usage.

## Important limitations

TubeDesk depends on websites selected by the user. Availability, playback, account access, paid features, regional restrictions, and content policies are controlled by those third-party websites and are not controlled by TubeDesk.

## Disclaimer

TubeDesk is an independent desktop workspace. It is not affiliated with, endorsed by, sponsored by, or approved by any third-party website, media service, or platform.

All third-party product names, trademarks, content, and services belong to their respective owners.

## Store category suggestion

Productivity is the strongest category fit because the app's primary value is workflow organization, focus, shortcut management, and desktop window control. Entertainment may be considered only if screenshots and metadata emphasize media consumption rather than workspace productivity.

## Age rating notes

The app can display user-selected web content. Actual content available to users depends on the websites they choose, account settings, regional availability, and user activity.

## Privacy summary

TubeDesk does not collect, sell, share, or transmit personal data to the developer. If users sign in to a website inside TubeDesk, login is handled directly by that website inside the embedded browser session.
